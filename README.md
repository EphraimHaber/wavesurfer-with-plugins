# wavesurfer-with-plugins

Small Vite + React app that demos official [wavesurfer.js](https://wavesurfer.xyz/) plugins (regions, timeline, minimap, spectrogram, record, envelope, hover, zoom).

**Live demo:** [ephraimhaber.github.io/wavesurfer-with-plugins/regions](https://ephraimhaber.github.io/wavesurfer-with-plugins/regions)

Swap the last path segment for another plugin: `timeline`, `minimap`, `spectrogram`, `record`, `envelope`, `hover`, `zoom`.

## Run locally

```bash
cd wavesurfer-plugins
pnpm install
pnpm dev
```
